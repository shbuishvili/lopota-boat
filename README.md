# Lopota Boat Race (მინი თამაში)

მინი თამაში — ნავების შეჯიბრი Lopota Lake Resort & Spa-ის თემატიკით, დეპარტამენტების მიხედვით.

## გაშვება

1) გახსენი `lopota-boat-race/index.html` ნებისმიერ ბრაუზერში (Chrome/Edge).

## კონტროლი

- `Space` ან ეკრანზე `Paddle` — “გალიჩვა” / ბიძგი
- ტელეფონზე/ტაჩზე — უბრალოდ დააჭირე `Paddle`

## ლოგო

- ნაგულისხმევად გამოიყენება placeholder ლოგო (`assets/lopota-logo.svg`).
- შეგიძლია ატვირთო რეალური ლოგო თამაშის ეკრანიდან (`Upload logo`) — ჩაიმახსოვრებს ბრაუზერში (localStorage) და გამოიყენებს ნავზე/მაისურზე.

## კასტომიზაცია

- დეპარტამენტების სია და ფერები: `lopota-boat-race/game.js` (მასივი `DEPARTMENTS`)
